# 要求和建议
在使用 Yi-Anyun SDK for PHP 前请确保您的环境支持以下要求和建议。


## 要求
- 您必须使用 PHP 5.5.0 或更高版本。
- 使用 [Composer][composer] 并优化自动加载 `composer dump-autoload --optimize`


## 建议
- 安装 [cURL][cURL] 7.16.2 或更高版本
- 生产环境中不要使用 [Xdebug][xdebug]
